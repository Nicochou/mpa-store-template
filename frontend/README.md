# Client
## Theme installation
````
`npm run theme:simplex`
`npm run theme:flat`
`npm run theme:lux`
`npm run theme:minty`
`npm run theme:glossy`
```` 
## Installation
````
`npm install`
```` 
## Developpement
````
`npm start`
```` 
## Build
````
`npm build`
```` 
## Tests
````
`npm test`
```` 
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
